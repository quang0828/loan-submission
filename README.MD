# Nodejs server

## Project setup
```
yarn install
```

### Start server

Install `Mongodb`

Create change `.env` from `.env.sample`

```
yarn start
```

# Web

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development

Create and change `.env` from `.env.sample`

```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
